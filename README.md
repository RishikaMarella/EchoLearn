## 🧠 About

**EchoLearn** is an AI-powered, voice-enabled personal study companion that delivers structured, interactive lessons on **any topic** — anytime, anywhere.

A student simply **speaks or types** what they want to learn. EchoLearn instantly:
- Generates a complete structured lesson using **Gemini AI**
- Reads it aloud in a natural human voice using **Murf AI**
- Recommends curated **YouTube videos**
- Tests understanding with an **auto-generated quiz**
- Saves **notes and lesson history**

> 💡 *No more juggling 5 different apps. EchoLearn brings everything into one beautiful platform.*

## ❌ The Problem

Students across the world face these challenges every single day:

| Challenge | Impact |
|---|---|
| 💸 Personal tutors are expensive | Quality education is unaffordable for most |
| 📚 Learning is passive & one-way | Students read & watch but don't interact |
| 🔀 Resources are scattered | Students juggle YouTube, notes, quiz apps separately |
| ⏰ Classrooms are time-restricted | Learning stops outside school hours |
| 🌍 Quality education is unequal | Not everyone has access to good teachers |
| 🔇 No instant feedback | Students don't know if they truly understood |

<br/>

## ✅ The Solution

**EchoLearn** solves every single one of these problems:

| Problem | EchoLearn's Solution |
|---|---|

| Passive learning | Voice input + AI voice output = interaction |
| Scattered resources | Lessons + Videos + Quiz + Notes in one app |
| Time restricted | Available anywhere, anytime, on any device |
| Unequal access | Free and accessible to every student globally |
| No feedback | Instant auto-generated quiz after every lesson |

## ✨ Features

<br/>

### 🔐 Authentication
- One-click **Google OAuth** sign-in
- Guest login option for instant access
- Secure session management across pages

<br/>

### 🐾 Personal Study Buddy
- Choose from **9 animated animal companions**
- Beautiful slide-based buddy picker with arrows
- Your buddy stays with you through every lesson

<br/>

### 🎙️ Voice Learning *(Core Feature)*
- **Speak your topic** — OpenAI Whisper transcribes it instantly
- **Listen to your lesson** — Murf AI reads it in a natural human voice
- Browser speech synthesis as automatic fallback

<br/>

### 📚 AI-Powered Structured Lessons
Every lesson generated by **Google Gemini AI** includes:
- 📖 Simple Explanation
- 🔑 3 Key Points
- 🌍 Real Life Example
- 💡 Fun Fact
- 📝 Quick Summary

<br/>

### 🧠 Interactive Quiz
- 3 auto-generated multiple choice questions per lesson
- Instant scoring with correct answer revealed
- Retry option for repeated practice

<br/>

### 📺 YouTube Video Cards
- Real videos fetched via **YouTube Data API**
- Displayed as beautiful clickable cards with thumbnails
- Auto-matched to your exact lesson topic

<br/>

### 📝 Notebook & History
- Quick note-taking section inside every lesson
- Full notebook accessible from the navbar
- Complete lesson history with date and subject saved

<br/>

### 💬 Follow-Up Chat
- Ask any follow-up question after your lesson
- Powered by Gemini AI
- Murf AI voice available for answers too

<br/>

### 🎨 Beautiful UI
- Fluid animated gradient background
- Glassmorphism card design
- Smooth page-by-page transitions
- Motivational quote widget

## 🚀 Getting Started

### Prerequisites

Make sure you have these installed on your system:

```bash
python --version    # Should be 3.10 or above
pip --version       # Should be installed with Python
git --version       # For cloning the repo
```

<br/>

### Step 1 — Clone the Repository

```bash
git clone https://github.com/RishikaMarella/EchoLearn.git
cd EchoLearn
```

<br/>

### Step 2 — Install All Dependencies

```bash
pip install -r requirements.txt
```

<br/>

### Step 3 — Create Your `.env` File

```bash
# Create a new file called .env in the project root
# Add the following lines with your actual API keys:

GEMINI_API_KEY=your_gemini_api_key_here
YOUTUBE_API_KEY=your_youtube_api_key_here
MURF_API_KEY=your_murf_api_key_here
GOOGLE_CLIENT_ID=your_google_client_id_here
GOOGLE_CLIENT_SECRET=your_google_client_secret_here
```

<br/>

### Step 4 — Run the Application

```bash
streamlit run app.py
```

<br/>

### Step 5 — Open in Browser

```
http://localhost:8501
```

🎉 EchoLearn is now running on your machine!

## 🔑 Environment Variables

| Variable | Where to Get It | Cost |
|---|---|---|
| `GEMINI_API_KEY` | [aistudio.google.com](https://aistudio.google.com) → Get API Key | ✅ Free |
| `YOUTUBE_API_KEY` | [console.cloud.google.com](https://console.cloud.google.com) → YouTube Data API v3 | ✅ Free |
| `MURF_API_KEY` | [murf.ai](https://murf.ai) → API Section | ✅ Free Tier |
| `GOOGLE_CLIENT_ID` | [console.cloud.google.com](https://console.cloud.google.com) → OAuth 2.0 Credentials | ✅ Free |
| `GOOGLE_CLIENT_SECRET` | Same location as Client ID | ✅ Free |

> ⚠️ **Security Notice:** Your `.env` file is listed in `.gitignore` and will **never** be pushed to GitHub. Keep your keys safe and never share them publicly.

## ☁️ Deployment

EchoLearn is deployed for free on **Streamlit Cloud**.

To deploy your own version:

```
1. Push your code to GitHub
2. Go to share.streamlit.io
3. Sign in with GitHub
4. Click "New App"
5. Select your EchoLearn repository
6. Set main file path as: app.py
7. Click "Advanced Settings" → add your API keys under Secrets
8. Click "Deploy!"
```

Your live app will be available at:
```
https://your-chosen-name.streamlit.app
```
